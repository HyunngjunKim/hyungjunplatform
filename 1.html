h1 은 h6까지 제목에 관한 태그

Hypertext markup language (HTML) is the standard markup language for creating web pages and web applications.




HTML은 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타내는 것뿐만 아니라 링크이다. 이렇게 이미지 삽입

이렇게 글자크기수정 HTML은 웹 페이지 콘텐츠 안의 꺾쇠 괄호에 둘이다. 와 본문과 그 밖의 항목의 외관과 배치를 정의하는 CSS 같은 스크립트를 포함하거나 불러올 수 있다.

HTML과 CSS 표준의 공동 책임자인 W3C는 명확하고 표상적인 마크업을 위하여 CSS의 사용을 권장한다
